🚀 Setup

Quick Install

curl -sL https://raw.githubusercontent.com/CCAgentOrg/picorouter/main/install.sh | bash

Manual

git clone https://github.com/CCAgentOrg/picorouter
cd picorouter
pip install -r requirements.txt
cp config.example.yaml picorouter.yaml
python picorouter.py serve

Docker

docker run -p 8080:8080 ccagentorg/picorouter

API Keys

PicoRouter supports multiple backends for storing provider API keys:

1. Environment Variables (Default)

export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
export GROQ_API_KEY="gsk_..."
export KILO_API_KEY="..."

2. .env File

export PICOROUTER_SECRETS_BACKEND=dotenv

# Create .env file
echo 'OPENAI_API_KEY="sk-..."' > .env
echo 'ANTHROPIC_API_KEY="sk-ant-..."' >> .env

3. Vaultwarden (Self-hosted Bitwarden)

export PICOROUTER_SECRETS_BACKEND=vaultwarden
export VAULTWARDEN_TOKEN="your-session-token"

# CLI to manage
python picorouter.py secrets list

4. Encrypted File

export PICOROUTER_SECRETS_BACKEND=encrypted
export PICOROUTER_SECRETS_PASSWORD="your-password"

CLI Commands

# Generate example config
python picorouter.py config --example

# Start server
python picorouter.py serve --profile chat

# Tailscale support
python picorouter.py serve --host tailscale
python picorouter.py serve --show-ips

# Manage API keys (for PicoRouter itself)
python picorouter.py key add -n mykey --rate-limit 60

# Manage provider secrets
python picorouter.py secrets list
python picorouter.py secrets set --provider openai --key "sk-..."
