🚀 Setup

One-Command Install

curl -sL https://raw.githubusercontent.com/CCAgentOrg/picorouter/main/install.sh | bash

Step-by-Step

1. Clone

git clone https://github.com/CCAgentOrg/picorouter.git
cd picorouter

2. Install

pip install -r requirements.txt

Or with uv:

uv pip install -r requirements.txt

3. Configure

python picorouter.py config --example
# Edit picorouter.yaml with your API keys

4. Run

python picorouter.py serve

Connect Apps

OpenClawbaseUrl: http://localhost:8080/v1
VSCodeopenai.apiBaseUrl: http://localhost:8080/v1
Claude CodeOPENAI_API_BASE=http://localhost:8080/v1
