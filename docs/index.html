🧩 PicoRouter

Your personal OpenRouter — lean, local-first, and you control everything.

Quick Start

curl -sL https://raw.githubusercontent.com/CCAgentOrg/picorouter/main/install.sh | bash

Supported Providers

🖥️ Local

Ollama, LM Studio

☁️ Cloud

Kilo, Groq, OpenRouter

🤖 Major

OpenAI, Anthropic, Google, Mistral

🔗 Aggregators

Together, DeepInfra, Fireworks

Features

🌊 Local-First

Free, unlimited, private

🔄 Auto-Failover

Seamless cloud fallback

🧠 Content Routing

Auto-detect code/reasoning

🔐 Multi-Key Auth

Per-key limits & caps

📊 Usage Tracking

Tokens, cost, logs

🐳 Docker

Alpine ~50MB

Interfaces

  • API: REST at /v1/chat/completions
  • CLI: serve, chat, logs, key commands
  • Web UI: picorouter-web

Run

docker run -p 8080:8080 ccagentorg/picorouter
