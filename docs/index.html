🧩 PicoRouter v0.0.1

Your personal OpenRouter — lean, local-first, AI model router.

Quick Start

curl -sL https://raw.githubusercontent.com/CCAgentOrg/picorouter/main/install.sh | bash

Features

🌊 Local-First

Ollama, LM Studio — free, unlimited, private

🧠 Content Routing

Auto-detect code, reasoning, prompts

🏷️ Virtual Providers

privacy, free, fast, sota

📡 Header Routing

X-PicoRouter-* headers

🔐 Multi-Key Auth

Per-key limits & profiles

💾 Storage

JSONL, SQLite, Turso

🔑 Secrets

Env, Vaultwarden

🌐 Tailscale

VPN-friendly

Providers

🖥️ Local

Ollama, LM Studio

🔗 More

Mistral, Cohere, Together...

Run

# Docker
docker run -p 8080:8080 ccagentorg/picorouter

# Binary
./picorouter serve

# pip
pip install picorouter
picorouter serve

API

curl -X POST http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"messages":[{"role":"user","content":"Hi"}]}'
