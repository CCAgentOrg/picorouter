⚙️ Configuration

Config File

Location: picorouter.yaml

Profile Example

profiles:
  chat:
    local:
      provider: ollama
      endpoint: http://localhost:11434
      models: [llama3]
    
    cloud:
      providers:
        kilo:
          models: [minimax/m2.5:free]
        openrouter:
          models: [openrouter/free]
    
    routing:
      - if: short_prompt
        use_local: true
      - if: contains_code
        use_local: true
        models: [codellama]
    
    yolo: false

Environment Variables

export KILO_API_KEY="sk-..."
export GROQ_API_KEY="gsk_..."
export OPENROUTER_API_KEY="sk-or-..."

Database (Optional)

database:
  turso_url: "libsql://your-db.turso.io"

CLI Commands

python picorouter.py serve --profile coding
python picorouter.py chat -m "Hello"
python picorouter.py logs -s
python picorouter.py config --example
